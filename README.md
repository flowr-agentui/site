# Flowr Agentic Website Builder

This is a client-side AI website builder powered by LangChain and Google Gemini.

## Features
- **Interactive Chat Agent**: Discuss your website requirements with an AI.
- **Dynamic Preview**: See your website generated in real-time within an iFrame.
- **Client-Side**: Requires your Gemini API key to run directly in the browser.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open the browser (usually http://localhost:5173).

4. Enter your Google Gemini API Key when prompted.
